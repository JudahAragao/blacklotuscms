# Sithemep Specification

## Description
Geracao automatica de XML sithemep baseado em configuracoes de inclusao de PostTypes.

## Requirements
- **REQ-01:** Geracao de XML sithemep
- **REQ-02:** Inclusao configuravel por PostType
- **REQ-03:** Deletion de posts with noIndex
- **REQ-04:** Apenas posts published

## Constraints
- **C01:** PostTypes inclusao via setting sithemep_post_types
- **C02:** Default: page e post

## Dependencies
- **Depends on:** Post Management, Settings
- **Blocks:** NONE
- **Related to:** SEO
